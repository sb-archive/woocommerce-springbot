== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woocommerce-springbot` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Springbot WP Plugin -> Settings page to enter Springbot user credentials
4. If the credentials are valid, you will get a notification banner stating you are connected
5. Click on the banner to begin using Springbot!

== Frequently Asked Questions ==

= What happens if Springbot has a new version? =

We will notify your Wordpress site that a new version exists, and you'll have the opportunity to accept or decline those changes.

= My credentials failed -- what should I do? =

Make sure your credentials are correct and entered as they appear on Springbot. Do not copy and paste credentials. Type them manually. If they are entered correctly, then contact support@springbot.com for help!
